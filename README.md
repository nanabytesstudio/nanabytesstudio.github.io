# Nana Bytes

![Nana Bytes](./assets/banner.jpg)

## About

Nana Bytes is an independent mobile app studio focused on creating simple, useful and privacy-friendly applications.

We believe software should be:

* Simple to use
* Fast and reliable
* Respectful of user privacy
* Built with attention to detail

## Applications

Current and upcoming applications developed by Nana Bytes are available on Google Play.

Visit the website for more information:

https://nanabytesstudio.github.io

## Privacy First

Our applications are designed with a privacy-first approach.

* No mandatory accounts
* No unnecessary data collection
* Local-first storage whenever possible
* Minimal permissions
